# Weather & Air Quality Analysis Dashboard

## Project Overview

This Power BI dashboard provides insights into weather conditions and air quality metrics for different locations. The dashboard combines current weather, air quality indicators, and forecast data to support environmental monitoring and analysis.

## Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel
- Data Modeling

## Dataset

The project uses multiple Excel datasets:

1. Current Weather & Air Quality Data
2. Daily Forecast Data
3. Hourly Forecast Data

## Dashboard Features

### Weather Monitoring
- Current Temperature
- Humidity
- Wind Speed
- Visibility
- Pressure
- UV Index
- Precipitation

### Forecast Analysis
- 7-Day Temperature Forecast
- Rain Probability Analysis
- Sunrise & Sunset Information

### Air Quality Monitoring
- Air Quality Index (AQI)
- PM10
- PM2.5
- O3 (Ozone)
- SO2
- NO2
- CO

### Interactive Features
- Location Selection
- Dynamic KPI Cards
- Forecast Trends
- Air Quality Visualization

## Data Model

The dashboard follows a star-schema style model with relationships between:

- Location Table
- Current Data Table
- Daily Forecast Table
- Hourly Forecast Table

## Key Insights

- Track current weather conditions.
- Monitor air quality levels.
- Analyze temperature trends.
- Evaluate rainfall probability.
- Compare environmental conditions across locations.
